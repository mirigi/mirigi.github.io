Mirigi turns the **smart home into a natural extension of condominium life**. The same app residents already use for visitors, deliveries, reservations and service requests now **drives in-unit automation** — *one unified app*. Built on top of [BeoLiving Intelligence, the smart home controller by Khimo and Bang & Olufsen](https://www.khimo.com/#products-bli), **every device the BLI controller can reach** becomes available from a phone, tablet or any Internet-enabled device, **anywhere in the world**.

Inside every apartment, Mirigi **pulls the automation configuration directly from the BLI controller** and lays it out **by location (zone) and controller (group)**, so each command appears exactly where the installer placed it. Whatever the installer exposed — signals and macros — all lives on a *single screen*. Installer-defined signals are rendered as **one-tap buttons**, and installer-defined macros run multi-command sequences in a single tap — and because commands route through Mirigi's backend, the unit responds whether the resident is on the couch or halfway around the world.

For **staff and building operators**, the model is just as clean. Because each unit's command list is read straight from its BLI controller, there is **no per-resident interface to maintain** inside Mirigi: installers configure the smart home *once* on the controller and Mirigi reflects it automatically. The result is **smart-home access** unified with the rest of the resident experience, and onboarding for new residents that is **quick, consistent and effortlessly premium**.
